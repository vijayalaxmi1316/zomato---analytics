# Zomato Analytics Project 
## Project Overview

This project focuses on analyzing Zomato restaurant data using Microsoft Excel to extract meaningful business insights. The analysis helps understand restaurant distribution, customer ratings, cuisines, and city-level performance. Excel features such as formulas, pivot tables, and charts are used to convert raw data into actionable insights.

## Objectives

Analyze restaurant distribution across cities and countries
Understand customer rating patterns
Identify popular cuisines and high-performing restaurants
Perform city-wise and cuisine-wise analysis
Create interactive dashboards using Excel

## Dataset Description

The dataset includes the following information:
Restaurant Details: Restaurant Name, City, Country
Cuisine Information: Cuisines Offered
Ratings: Aggregate Rating, Rating Count
Cost Details: Average Cost for Two
Service Details: Online Delivery, Table Booking
Data cleaning and preprocessing were done in Excel before analysis.

## Tools & Techniques Used

Microsoft Excel
Data Cleaning
Excel Formulas (IF, COUNTIFS, SUMIFS, AVERAGEIFS)
Pivot Tables & Pivot Charts
Conditional Formatting
Dashboard Creation

## Key KPIs

Total Number of Restaurants
Average Restaurant Rating
City-wise Restaurant Count
Cuisine-wise Restaurant Distribution
Rating Distribution

## Analysis & Insights

Most restaurants fall within mid-range ratings

## Key Features

Cuisine-based performance insights
Rating distribution using charts

Well-structured Excel dashboard

## Conclusion

The Zomato Analytics project demonstrates how Excel can be effectively used as a powerful data analytics tool. By leveraging Excel formulas, pivot tables, and visualizations, raw restaurant data is transformed into valuable insights. The analysis highlights trends in customer ratings, popular cuisines, and city-wise restaurant distribution. These insights can help stakeholders understand market demand, improve service strategies, and make informed business decisions. This project showcases strong foundational skills in data analysis, Excel reporting, and business intelligence, making it an excellent portfolio project for aspiring data analysts.
